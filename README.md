## üöÄ Getting Started

### 1. Prerequisites

- **Node.js** (v18+)
- **Julia** (v1.9+)
- **NVIDIA GPU** (Optional, for CUDA acceleration)

### 2. Installation

```bash
# Install Node.js dependencies
npm install

# Initialize Julia environment
npm run julia:install
```

### 3. Launch the System

```bash
# Start the API Gateway
npm start

# In a separate terminal, launch the IDE
npm run ide
```
